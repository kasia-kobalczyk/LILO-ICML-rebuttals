# Supplementary materials for LILO rebuttals

This repository contains supplementery materials (Figures and Tables) for the ICML 2026 rebuttals of the paper: LILO: Bayesian Optimization with Natural Language Feedback.

Contents: 
- NAS-Bench-201.pdf -- New becnhmark environment with two test problems based on NAS-Bench-201
- Tex-summarization.pdf -- A new example experiment with high dimensional and unstructured outputs (i.e., textual summaries of news articles) based on the CNN/DailyMail dataset
- Related-works.pdf -- A table outlining the novelty of LILO and positioning it against the existing families of related works: preferential BO, LLM-Enhanced BO, Language-based peference elicitation
